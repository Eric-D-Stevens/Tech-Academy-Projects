This text document is a description contents of the SQL Final Project for Tech
Academy

FILES

  Table_Creation.sql: This file contains SQL Server code that builds the
  topology of the database. All tables needed are added and the relationsips
  betwen those tables are established.

  Table_Insertions.sql: This file is where the raw data is input into the 
  tables. All information contained in the tables throughout the project 
  is entered here and remains static in all other parts of the prject.

  Stored_Procedures.sql: This is the file where the stored procedures are
  defined. They are labeled by number and comented with the function they
  enable that was provided by Tech Acedemy.
